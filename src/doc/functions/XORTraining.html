Form

w = XORTraining(a,b,c,w,n,eta)

Description

Train the XOR net with back propagation derived for a tanh activation function. Includes a demo with random initial weights.

Inputs

a    (1,4) Input 1
b    (1,4) Input 2
c    (1,4) Output
w    (9,1) Weights and biases
n    (1,1) Number of iterations through all 4 inputs
eta  (1,1) Training weight

Outputs

w    (9,1)  Weights and biases
