CONE Compute the vertices for a cone.

The cone points along +z

Form:

[v, f] = Cone( r, h, n )

Inputs

r            (1,1) Base radius
h            (1,1) Height
n            (1,1) Number of divisions

Outputs

v            (:,3) Vertices
f            (:,3) Faces
