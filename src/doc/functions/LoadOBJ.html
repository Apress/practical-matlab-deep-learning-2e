Form:

g = LoadOBJ( file, path, kScale )

Description

LoadOBJ( 'myFile.obj' )

LoadOBJ will open myFile.obj. If you want to scale the units of the model, for example from vertex units to inches, you can define the scale factor using kScale.

If you call the function with no outputs, it will draw the model. If you call the function with an empty filename you can select a file using the dialog box. For a demo type LoadOBJ.

Inputs

file        (1,:) Filename
path        (1,:) Path to the file
kScale      (1,1) Optional scale factor of the model

Outputs

g           (1,1) Data structure
                  .name       (1,:)	Model name
                  .component	(:) 	Component structure
                  .radius     (1,1)
