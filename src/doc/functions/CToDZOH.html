Form

[f, g] = CToDZOH( a, b, T )

Description

Continuous to discrete transformation using a zero order hold. Discretize using a matrix exponential.

Given

.
x = ax + bu

Find f and g where

x(k+1) = fx(k) + gu(k)

Inputs

a  (n,n)  Continuous plant  matrix
b  (n,m)  Input matrix
T  (1,1)  Time step

Outputs

f  (n,n)  Discrete plant
g  (n,m)  Discrete input

See also

expm

References: Van Loan, C.F., Computing Integrals Involving the Matrix

             Exponential, IEEE Transactions on Automatic Control
             Vol. AC-23, No. 3, June 1978, pp. 395-404.
