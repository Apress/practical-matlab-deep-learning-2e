Deep Learning with MATLAB

This is the companion code to the Practical MATLAB Deep Learning book from Apress. All code referenced in the book is included in its entirety. The documentation has been automatically generated by publishing the functions and demos.

Requirements

The book and code were developed using Release R2021b of MATLAB. Some features may not function in earlier versions. This code requires the Deep Learning and Reinforcement Learning toolboxes throughout the chapters. Some chapters require additional MATLAB toolboxes.

  • Chapter 7 - Instrument Control toolbox
  • Chapter 9 - Image Processing toolbox
  • Chapter 11 - AlexNet and GoogleNet
  • Chapter 15 - Optimization toolbox

The installation of AlexNet and GoogleNet can be performed without additional licenses through the Add-On explorer. MATLAB will prompt you to install them when you run a funciton or script that requires them.

In order to run the music examples in Chapter 14, Generative Modeling of Music, you must install a Midi support package available on Github. For information please visit the site:

  • https://github.com/miditoolbox/
